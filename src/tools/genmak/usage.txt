Usage: %s [OPTIONS] LIBRARY_NAME SOURCES...

This utility generates a makefile for building a provider module.

OPTIONS:
    -V      -- print the version.
    -h      -- print this message.
    -f      -- overwrite the Makefile if it already exists.
    -C      -- generate a CMPI provider module (rather than Pegasus C++).
    -r      -- generate regmod target. (Experimental)
