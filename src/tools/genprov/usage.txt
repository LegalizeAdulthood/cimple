Usage: %s [OPTIONS] CLASSNAME

This utility generates the skeleton code for a provider. It generates two
files: <CLASSNAME>_Provider.h and <CLASSNAME>_Provider.cpp. It will only
overwrite existing files when the -f (force) option is used.

OPTIONS:
    -I      -- add to MOF include path (can be repeated).
    -M file -- add to list of MOF files to be parsed (can be repeated).
    -f      -- force overwrite of existing files (be careful).
    -m      -- generate the module file as well.
    -v      -- print the version.
    -h      -- print this message.
