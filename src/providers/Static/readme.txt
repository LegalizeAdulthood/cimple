
Summary:
========

This directory illustrates how to build a provider that uses static cimple
and cimplecmpiadap libraries.

Steps:
======

    1.  Build the static version of cimplecmpiadap (called cimplecmpiadap_s).
        Change to cimple/src/cmpi/adapter_s and type "make".

    2.  Next return to this directory and type "make".
